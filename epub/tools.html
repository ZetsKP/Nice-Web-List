Useful Tools for Readers

ðŸ’» For Desktop User

  • Koodo Reader : Buka dokumen secara online agar Chrome Extension dapat berfungsi terhadap dokumen.
  • PDF Reader : UX nya terbaik. Extension dapat berjalan lancar, di render seperti web pada umumnya.
  • EPUB Reader : UX nya terbaik. Namun Extension tidak dapat berjalan akibat "Sandboxing" [Baca].
  • EPUB to HTML : Convert EPUB ke 1 halaman HTML, selanjutnya dapat di hosting di GitHub Pages.
  • 1 Click Translator : Terjemahkan text cukup dengan select text tertentu. Keunggulan : Cepat dan fleksibel.
  • TransOver : Terjemahkan text cukup dengan select text tertentu + hover. Keunggulan : Detail terjemah kata.
  • Google Translate : Manage Extension -> Keyboard Shorcuts -> Google Translate, lalu isi dengan Ctrl+B.
  • Copyfish Free OCR Software : Convert image menjadi text yang dapat di copy/manipulasi/translate.
  • Natural Reader Text to Speech : Baca text sebagai audio dengan Select text maupun Full Page.
  • Web & PDF Highlighter LINER : Highlight text penting di website maupun file PDF/EPUB anda.
  • Mercury Reader : Bersihkan tampilan website dari distraksi, fokuskan hanya ke bagian tulisan/arikel saja.
  • Unlimited Medium : Membuka akses artikel premium di Medium secara gratis hanya dengan satu klik.
  • Click to Remove Element : Hapus section iklan yang mengganggu dengan hanya 1-klik-an saja.

  • EPUB to HTML (Back) : Baca beberapa dokumen yang sudah di convert dari EPUB ke HTML.
  • PDF Reader : Buka PDF secara online. UX nya terbaik, bahkan untuk versi mobile sekalipun.

  • Textcopy (for Android) : Copy, Paste dan Translate apa pun di layar Android anda dengan mudah.
  • T2S Text to Voice (for Android) : Baca text sebagai audio dengan Select text di Android anda.
