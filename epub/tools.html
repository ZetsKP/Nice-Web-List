Reader: PDF

  1. PDF.js : Buka PDF secara online. Bisa translate kalimat dengan memakai 1 Click Translator.
  2. Koodo Reader : Buka PDF, EPUB & MOBI secara online agar Extension dapat digunakan.
  3. Kami Extension : Buka PDF secara online. Bisa translate dengan Extension. Ada fitur Highlight.

Reader: EPUB

  1. Immerse Reader : Buka EPUB secara online. Bisa translate full page via Chrome Browser.
  2. ePubViewer [Swipe Mode] : Buka EPUB secara online. Bisa translate full page via Chrome Browser.
  3. Bibi EPUB Reader : Buka EPUB secara online. Gak bisa translate, namun UX nya bagus.
  4. EPUB to HTML : Convert EPUB ke HTML, selanjutnya anda dapat hosting di GitHub Pages.

Extension: Translator

  1. 1 Click Translator : Terjemahkan text cukup dengan select text tertentu. Cepat dan fleksibel.
  2. TransOver : Terjemahkan text cukup dengan select text tertentu/hover. Bisa terjemah detail kata.
  3. Google Translate : Manage Extension -> Keyboard Shorcuts -> Google Translate, isi Ctrl+B.
  4. Copyfish Free OCR Software : Convert image ke text yang dapat di copy/manipulasi/translate.

Extension: Summarizer

  1. EndNote Click : Akses cepat sekali klik ke jutaan makalah penelitian. Bisa generate format PDF.
  2. Scholarcy : Ringkas makalah penelitian hanya dengan satu klik. Bisa membaca format PDF.
  3. GetDigest : Highlight informasi terpenting dari setiap artikel web secara cepat (Summary).
  4. Frase Article Summarizer : Ringkas artikel dalam sekali klik. Situs Medium.com, berita, dll.
  5. Crammer : Analisis text dalam web menggunakan AI. Temukan "Keypoint" dengan cepat.
  6. â‡¨ Temukan beberapa Text Summarizing Tools terbaik lainnya yang tersedia. [Open]

Extension: Speech & More

  1. Natural Reader Text to Speech : Baca text sebagai audio dengan select text maupun full page.
  2. Web & PDF Highlighter LINER : Highlight text penting dari website/PDF/EPUB anda.
  3. Mercury Reader : Bersihkan tampilan website, fokuskan hanya ke bagian tulisan/arikel saja.
  4. Unlimited Medium : Membuka akses artikel premium di Website Medium.com secara gratis.
  5. Click to Remove Element : Hapus section iklan yang mengganggu dengan hanya 1 klik.

Ebook Reader

  1. PDF.js : Buka PDF secara online. Bisa translate kalimat via Screen Translate App.
  2. ePubViewer [Swipe Mode] : Buka EPUB secara online. Bisa translate full page via Chrome.
  3. Immerse Reader : Buka EPUB secara online. Bisa translate full page via Chrome. UX nya buruk.

Usefull Apps

  1. GetDigest : Highlight informasi terpenting dari setiap artikel di website. Bisa juga upload file.
  2. IvyPanda Text Summarizer : Paste artikel, esai atau teks lainnya dan generate ringkasan secara cepat.
  3. Screen Translate [for Android] : Menerjemahkan teks secara langsung di layar Android anda.
  4. Textcopy [for Android] : Copy, Paste dan Translate apa pun di layar Android anda dengan mudah.
  5. T2S Text to Voice [for Android] : Baca text sebagai audio dengan Select text di Android anda.
